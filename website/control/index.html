Access Flow

Three clean steps to open the live dashboard.

The Control Center is the web dashboard for Vanguard. It gives server operators a cleaner place to review welcome flow, guard settings, moderation routing, and per-server configuration without relying only on slash commands inside Discord.

Step 1

Enable the web surface

Turn on the control center in your Vanguard environment so the bot can serve the dashboard securely.

Step 2

Configure authentication

Set up Continental ID auth and make sure the Continental account used for sign-in is linked to the Discord account that manages the server.

Step 3

Open your live route

Visit your deployment at /control/ to manage welcome flow, guard settings, and operations routing in one place.

No public shared dashboard is hosted here. This page explains access only. The real control center lives inside the Vanguard instance you run.
