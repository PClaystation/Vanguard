FAQ

Common questions about the Vanguard Discord bot.

What kind of Discord bot is Vanguard?

Vanguard is a Discord moderation bot built for server safety and operations. It includes moderation commands, anti-raid guard presets, reminders, structured voting, and an optional web Control Center for server management.

Can Vanguard help with anti-raid protection?

Yes. Vanguard includes guard presets plus advanced controls for join floods, mention bursts, duplicate messages, link spam, and automated mitigation steps such as slowmode, message deletion, and optional timeouts.

How do you open the Control Center?

The public Control Center page explains the access flow. The actual dashboard is served by your own Vanguard deployment after you enable the embedded web server and configure Continental ID authentication.

Does Vanguard support voting and operator workflows?

Vanguard supports polls, structured vote flows, ballot tracking, reminders, moderation case logging, and configuration surfaces designed for operators who need clearer server administration.
