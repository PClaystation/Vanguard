Guard System

Layered anti-raid protection with automated response controls.

Detection Signals

Multi-Signal Threat Tracking

  • New-account message bursts
  • Join floods in short windows
  • Mention spam and ping bursts
  • Duplicate-message raids
  • Link and invite spam waves

Automated Defense

Fast, Graduated Mitigation

  • Severity-aware trigger logic
  • Scoped slowmode in active channels
  • Optional auto-timeout on trigger
  • Optional trigger-message deletion
  • Cooldown + suppression protection
